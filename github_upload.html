Upload ke GitHub (Cara Instan — Lewat Browser)

⚠
Tidak perlu Token API! Cara ini menggunakan halaman web GitHub secara langsung. Cukup login ke akun GitHub Anda di browser, lalu ikuti langkah di bawah.
1
🔒 Login ke GitHub
Buka github.com dan pastikan sudah login dengan akun Anda.
2
📁 Buka Repository & Halaman Upload
Masuk ke repository Anda, klik tombol Add file → pilih Upload files. Atau langsung buka URL dengan format:
https://github.com/[username]/[repo]/upload/[branch]
📸 Tampilan tombol "Add file" ada di sebelah tombol hijau "Code"
3
📥 Drag & Drop File ke Kotak Upload
Di halaman upload GitHub, terdapat kotak besar dengan tulisan "Drag files here to add them to your repository".

Pilihan A — Upload file langsung: Tarik semua file (index.html, data/*.json, script.js, dll.) dari File Explorer / Finder langsung ke kotak tersebut.

Pilihan B — Upload dari ZIP: Klik tombol Download .ZIP di atas, lalu ekstrak ZIP-nya, kemudian drag semua isinya ke kotak GitHub.
⚠ GitHub tidak mendukung upload file ZIP langsung — file harus di-ekstrak dulu
4
✅ Isi Commit Message & Klik "Commit changes"
Setelah file terupload ke kotak, scroll ke bawah. Isi Commit message (contoh: feat: upload website files), pilih branch tujuan, lalu klik tombol hijau Commit changes.
✓ File Anda akan langsung muncul di repository setelah commit berhasil
💡
Untuk website statis (GitHub Pages): Setelah upload selesai, aktifkan GitHub Pages di Settings → Pages → Source → main branch. Website Anda akan live di https://[username].github.io/[repo]
